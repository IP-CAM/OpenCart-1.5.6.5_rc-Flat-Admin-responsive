How To Install

  1. If you already had working opencart store so take a full backup for all its folders, files and database then skip the first 2 steps and jump to step 3.
    If it's your first time to install a new opencart store so you must first go to Official Opencart Site and download opencart default theme that compatable with our theme version from here Download default Opencart theme
  2. Install the default opencart theme that you downloaded.
  3. Now install vqmod if you didn't installed it before.
  4. Open any ftp program like Filezilla Client and login to your store by your ftp.
  5. Open from local window side this folder from your Themeforest downloaded package Theme_Upload (v.1.0) / 1.5.6.x
  6. Drag all folders and files from the local window and drop it on the remote server window.
  7. Now you need to activate valentino theme from admin panel so login to your admin panel and follow :
    Admin panel > system > setting > Edit > Store > Template: valentino > save
  8. Go to Admin panel > System > User > User Groups > Top Administrator
    Clcik Select All in poth (Access premission and Modify permission)
  9. Go to Admin panel > System > Design > Layout > insert
    Add new layout called all pages (case sensetive) then hit save button.
  10. Go to Admin panel > catalog > Extra Product tabs Then add at least one tab, not important to activate it inside any product but it's important to avoid any error
  11. Now Open your language file through your server :
    Go to : Catalog/language/*your language folder*/*english*.php, add these lines and save the changes : 
    // Custom
$_['text_advanced_search']             = 'Advanced search';
$_['text_save']                        = 'Save: ';
$_['text_sale']                        = 'SALE';
$_['text_menu']                        = 'Menu';
$_['text_see_details']                 = 'See Details';
$_['cat_title']    			   		   = 'Categories';
$_['text_links']    			   	   = 'Links';
$_['text_more']    			   	   	   = 'More';
$_['text_twitter']    			   	   = 'LATEST TWEETS';
$_['text_flickr']    			   	   = 'FLICKR STREAM';
  12. If arabic language Open your language file through your server :
    Go to : Catalog/language/*your language folder*/*arabic*.php, add these lines and save the changes : 
    // Custom
$_['text_advanced_search']             = 'البحث المتقدم';
$_['text_save']                        = 'خصم';
$_['text_menu']                        = 'القائمة الرئيسية';
$_['text_see_details']                 = 'التفاصيل';
$_['cat_title']    			           = 'الاقسام';
$_['text_links']    			   	   = 'روابط سريعة';
$_['text_more']    			   	   	   = 'المزيد';
$_['text_twitter']    			   	   = 'تويتر';
$_['text_flickr']    			   	   = 'فليكر';
$_['text_sale']    			   	       = 'خصم';
$_['text_blogs']    			   	   = 'اخبارنا';

Now you finished installing and activating our valentino theme, Go down to see how to manage your amazing valentino theme.
