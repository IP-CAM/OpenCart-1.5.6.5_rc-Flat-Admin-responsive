Installation

  • 1Make sure you have installed and running the correct version of OpenCart
  • 2It is a good practice to backup your system

  • 3Install Vqmod (Required)

    Marketshop makes use of plugins to add custom features. These plugins need Vqmod installed on your site.

    If, you already have install Vqmod. So, skip to this step.(VQMod installed required)
    Otherwise, get the latest release here http://code.google.com/p/vqmod/downloads/list. Download the lastest vqmod-x.x.x-opencart.zip file.

    Follow the instructions to install on your server http://code.google.com/p/vqmod/wiki/Install_OpenCart.


  • 4Unzip your downloaded "marketshop_opencart_theme_2.3.zip". There are subfolders structure:
  • 5Open the folder that corresponds to your OpenCart version and upload all the files from "FILES_TO_UPLOAD" to the root directory you have installed OpenCart in.
  • 6 Theme Activation
    • Login to your Opencart Admin Area
    • Go to System > Settings > Edit (your store)
    • Click on the Store tab. From the drop down listing of available themes select marketshop
  • 7 Theme Module Installation
    • Go to Extension > Modules > Marketshop Theme Control Panel and click on the 'install' link.
    • Then enable the 'Marketshop Theme Control Panel' module (click on the 'edit' link, change the status to 'Enabled' and click 'Save').

Featured, Latest, Bestsellers, Specials Modules

featured products, latest products, bestsellers, Specials modules, etc. adjust the image settings with the following recommended values:

For modules positioned Content Top or Bottom set the dimensions to 220x220
For modules positioned Column Left or Right set the dimensions to 50x50

 

Marketshop Product Tab

If you want to save space and show more products to your customers, install and enable the module “Product Tab”. It can display Featured Latest, Bestseller and Special products.

Product Tab module Only for Content Top or Bottom Position. don't Enabled in left or Right Position.

For modules positioned Content Top or Bottom set the dimensions to 220x220

 

Category Accordion

To display the whole list of categories in accordion format, install and enable the module “Category Accordion”.
It works the same way as the default Category module.

When use "Category Accordion" Module. So don't use "Category" default Module. Uninstall "Category" default Module

 

Marketshop - Latest Products By Category

Latest Products By Category module Only for Content Top or Bottom Position. don't Enabled in left or Right Position.

For modules positioned Content Top or Bottom set the dimensions to 220x220

 

Marketshop - Latest Products By Brand

Latest Products By Brand module Only for Content Top or Bottom Position. don't Enabled in left or Right Position.

For modules positioned Content Top or Bottom set the dimensions to 220x220

Advance Ajax Filter

Adv Ajax Filter filters products according to category, options, attributes, manufacturers, filters, tags and stock.

The Adv Ajax Filter module can be used on category type layouts (Category, Manufacturers, Search, Special) in the following positions:
  • Column Left
  • Column Right

Go to Extensions > Modules > Adv Ajax Filter > install & Edit
